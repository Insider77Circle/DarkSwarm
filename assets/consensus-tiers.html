📊 Consensus Flow Logic

Query Routing: When you submit a query, DarkSwarm automatically routes it through one or more consensus tiers based on the task requirements, network conditions, and your configured risk tolerance.

Escalation: If a Fast response cannot be verified or fails validation, the query automatically escalates to Standard. If Standard consensus fails, it escalates to Deep. Finally, Debate is used for adversarial hardening and novel problem-solving.

Self-Improvement: In Debate tier, nodes generate multiple candidate responses and argue about correctness. This process creates training data that improves your model over time, without any central server collecting data.
