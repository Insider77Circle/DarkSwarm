Title: DarkSwarm — P2P swarm for local LLM inference (llama.cpp, self-hosted)

Text:

Hi all — I'd like to share DarkSwarm, an open-source peer-to-peer compute
mesh for running local LLM inference across self-hosted nodes using llama.cpp.

Features:
- Run models locally and pool compute across trusted peers
- Privacy-focused: model weights and inputs stay under node operator control
- No central API fees; designed for self-hosted setups and edge deployments

Quickstart: see the project README and docs/QUICKSTART.md for a 2-command
example to get a node running.

If you're interested in decentralized AI, local LLMs, or privacy-preserving
inference, I'd love feedback and early testing. Links: https://github.com/Insider77Circle/DarkSwarm

Suggested communities: Hacker News (Show HN), r/LocalLLaMA, r/selfhosted,
Mastodon instances focusing on privacy/tech.
